# adventofcode2023

Learning Nim this year.

## Instructions

To compile and run:

```sh
nim c -r 01/main.nim

# pass in input file directly

nim c -r 01/main.nim < 01/input
```

Then once it's compiled, can simply run using:

```sh
./01/main

# pass in input file directly

./01/main < 01/input
```
